== Description ==

This plugin allows you to sell ebooks. It works with PayPal and has multiple currency option. 

This plugin adds a buy now button at the end of the book description. Book description can be written on posts. Simply add **"price"** to the custom value(without quotes) of the post. The price value should be without currency symbol.

You can choose curreny option from plugin settings page. 

For instructions visit http://www.revood.com/blog/using-bookshelf-2-0/

For suggestions and feedback visit http://www.revood.com/blog/bookshelf-2-0/

== Installation == 

1. Upload 'bookshelf' directory to /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Who can use this plugin? =

Any wordpress user willing to sell pdf books on their wordpress site.

= What are the supported payment options? =

Currently, Bookshelf supports only PayPal IPN

= How long is a buyer allowed to download the book? =

2 Days after the successfull payment.

= What are the currencies supported? =

All the currencies supported by PayPal
